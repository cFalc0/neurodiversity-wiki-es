# Glosario de Términos — Wiki de Neurodiversidad (ES)

Términos preferidos para la traducción y edición de artículos.
Este glosario es el lineamiento oficial para el cron de traducción y para ediciones manuales.

---

## Cómo usar este glosario

- La columna **Término preferido** es la que se usa en los artículos.
- La columna **Evitar** lista traducciones alternativas que existen pero que no usamos.
- La columna **Notas** explica el criterio o matiz cuando no es obvio.
